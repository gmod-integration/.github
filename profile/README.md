# Gmod Integration

Gmod Integration is the most advanced and feature-rich integration platform for connecting **Garry's Mod** servers with **Discord** communities.

Our open-source tools empower server owners to create seamless connections between their game servers and Discord guilds, enabling features like player verification, role synchronization, chat relay, and much more.
